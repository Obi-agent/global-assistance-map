# Global Assistance Partners Network Map

Interactive satellite map showing our worldwide network of medical and travel assistance service providers.

- 🟢 **Green markers** — fully signed agreement in place
- 🟠 **Orange markers** — agreement not yet signed

Built with [Leaflet.js](https://leafletjs.com/) and ESRI World Imagery satellite tiles with English place-name labels.
